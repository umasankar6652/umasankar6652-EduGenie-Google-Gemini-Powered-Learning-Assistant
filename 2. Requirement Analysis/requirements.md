# Phase 2: Requirement Analysis - EduGenie

## Software Requirements
- Operating System: Windows / macOS / Linux
- Language: Python 3.10+ (Current: Python 3.11.9)
- Web Framework: FastAPI (Uvicorn ASGI Server)
- AI Libraries: PyTorch, Hugging Face Transformers, Google Generative AI SDK
- Environment Management: python-dotenv (saving variables securely)
- Frontend: HTML5, CSS3 Variables (theme toggles, transitions), JavaScript (fetch APIs, localStorage caches)
- Markdown Parser: Marked.js CDN

## Hardware Requirements
- RAM: 8 GB minimum (16 GB recommended)
- CPU: Intel Core i5 / AMD Ryzen 5 / Apple Silicon M1 (runs local 77M model easily)
- Storage: 1.5 GB free space (for downloading LaMini-Flan-T5-77M model files)
