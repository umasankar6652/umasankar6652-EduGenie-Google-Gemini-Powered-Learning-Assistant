# Phase 1: Brainstorming & Ideation - EduGenie

## Early Ideas & Core Objectives
- Create a lightweight educational portal that helps students learn complex subjects easily.
- Support both cloud and local AI to solve the issue of expensive server API limits.
- Features must cover typical study patterns: asking questions, summarizing texts, generating tests for self-assessment, and roadmapping skills.
- Minimalist visual layout inspired by Google NotebookLM to emphasize content.

## Key Target Users
- Students preparing for exams (quick explanations, quizzes).
- Self-learners studying programming or complex topics (custom modules, projects).
- Educators looking to generate quick quizzes on textbook topics.
